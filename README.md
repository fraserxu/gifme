### Gifme
The easiest way to make and share a gif, right in the browser.

Live demo:
* [http://xvfeng.me/gifme/](http://xvfeng.me/gifme/)
* [[中文版]](http://xvfeng.me/gifme/cn.html)

#### Usage
```bash
$ cd gifme
$ npm i
$ npm i -g browserify
$ bin/build
$ open http://localhost:8000
```
